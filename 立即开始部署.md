## 🚀 现在开始部署（共约 25 分钟）

### 第一步：部署后端到 Railway（约 10 分钟）

1. **打开浏览器**，访问：https://railway.app
2. 点击 **"Login with GitHub"**
3. 点击 **"New Project"** → **"Deploy from GitHub repo"**
4. 选择仓库：**`gongzuoliu123`**
5. **配置服务**（关键步骤）：
   - 进入服务 → **Settings**
   - **Root Directory**: `backend`
   - **Build Command**: `npm install && npm run build`
   - **Start Command**: `npm start`
6. **添加环境变量**：
   - **Variables** 标签 → **New Variable**
   - `NODE_ENV` = `production`
7. **等待部署完成**（2-5 分钟）
8. **获取后端 URL**：
   - **Settings** → **Networking** → **Public Domain**
   - **复制这个 URL！** 📝

**详细步骤**：查看 `Railway部署说明.txt` 或 `部署操作指南.md`

---

### 第二步：部署前端到 Vercel（约 10 分钟）

1. **打开浏览器**，访问：https://vercel.com
2. 点击 **"Continue with GitHub"**
3. 点击 **"Add New Project"**
4. 选择仓库：**`jinghoor/gongzuoliu123`**
5. **配置项目**：
   - **Root Directory**: `frontend` ⚠️
   - **Framework**: Vite（自动检测）
6. **添加环境变量** ⚠️ **最重要！**：
   - **Name**: `VITE_API_BASE_URL`
   - **Value**: 粘贴你的 Railway 后端 URL（上一步复制的）
   - **Environment**: 全选 ✅
7. 点击 **"Deploy"**
8. **等待构建完成**（1-3 分钟）
9. **获取前端 URL** 🎉

**详细步骤**：查看 `Vercel部署说明.txt` 或 `部署操作指南.md`
